# 타입 단언
* src/ch10.ts
* 타입 단언의 규칙
    - 값 as 단언 <-- 단언식
    - A as B
        - A가 B의 슈퍼타입이거나
        - A가 B의 서브타입이어야 함
* 타입 단언은 업캐스팅과 다운캐스팅과는 다르게 눈속임 문법임(Non Null 같은 단언 사용시 주의 필요)
